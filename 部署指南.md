## 📦 部署步骤

### 步骤1：创建GitHub仓库

1. 打开 GitHub：https://github.com
2. 登录您的账号（用户名：`cherswang`）
3. 点击右上角 **"+"** → **"New repository"**
4. 填写信息：
   - **Repository name**：`house-rental-demo`（必须与此名称一致）
   - **Description**：`房屋租赁信息展示 - 360度全景看房`
   - **Visibility**：选择 **Public**（公开）
5. 点击 **"Create repository"**

### 步骤2：上传项目文件

#### 方法A：使用GitHub网页上传

1. 进入刚创建的仓库：`https://github.com/cherswang/house-rental-demo`
2. 点击 **"Add file"** → **"Upload files"**
3. 拖拽整个 `house-rental-demo` 文件夹的内容到上传区域
4. 在底部填写提交信息：
   ```
   Initial commit - 房屋租赁展示系统
   ```
5. 点击 **"Commit changes"**

#### 方法B：使用Git命令（推荐）

```bash
# 1. 进入项目目录
cd /Users/wanggang/Documents/trae_projects/house-rental-demo

# 2. 初始化Git仓库
git init

# 3. 配置用户名和邮箱（如果未配置）
git config user.name "cherswang"
git config user.email "your.email@example.com"

# 4. 添加所有文件
git add .

# 5. 提交
git commit -m "Initial commit - 房屋租赁展示系统"

# 6. 添加远程仓库
git remote add origin https://github.com/cherswang/house-rental-demo.git

# 7. 推送到GitHub
git push -u origin main
```

### 步骤3：启用GitHub Pages

1. 进入仓库设置：`https://github.com/cherswang/house-rental-demo/settings`
2. 滚动到 **"GitHub Pages"** 部分
3. 在 **"Source"** 下拉菜单中选择：
   - **Branch**: `main`
   - **Folder**: `/root`（保持默认）
4. 点击 **"Save"**
5. 等待1-2分钟，页面会显示：
   ```
   ✅ Your site is published at https://cherswang.github.io/house-rental-demo/
   ```

## ⚠️ 常见问题

### Q: 网站无法访问？

A: 请检查：
1. 确认GitHub Pages已启用（在仓库设置中查看）
2. 等待5分钟后再试（部署可能需要时间）
3. 检查URL是否正确：`https://cherswang.github.io/house-rental-demo/`

### Q: 图片无法显示？

A: 请检查：
1. 图片文件确实存在于 `assets/images/` 目录
2. 文件命名是否正确（使用英文，无特殊字符）
3. `script.js` 中的路径是否与实际文件一致

### Q: 全景图显示为黑色？

A: 请检查：
1. 全景图格式是否为等距圆柱投影（宽高比2:1）
2. 图片文件大小是否过大（建议不超过20MB）
