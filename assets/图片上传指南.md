1. 使用360度相机拍摄（推荐：RICOH THETA、Insta360等）
2. 使用鱼眼镜头拍摄多张照片后拼接（需要专业软件）
3. 从全景图素材网站购买或下载（需授权）

## 图片上传步骤

### 1. 将图片放入对应目录

```bash
# 打开图片文件夹
cd /Users/wanggang/Documents/trae_projects/house-rental-demo/assets/images

# 上传客厅图片
cp /你的图片路径/living-room.jpg ./living-room-1.jpg

# 上传餐厅图片
cp /你的图片路径/dining-room.jpg ./dining-room-1.jpg

# 以此类推...
```

### 2. 更新配置文件

上传图片后，需要在 `script.js` 中更新配置。找到 `const galleryImages = [` 部分，取消对应区域的注释并修改图片路径。

示例：
```javascript
{
    id: 7,
    src: 'assets/images/living-room-1.jpg',  // 修改为你的图片路径
    thumb: 'assets/images/living-room-1.jpg',
    category: 'living-room',
    title: '客厅全景',
    description: '宽敞明亮的客厅'
}
```

### 3. 刷新页面

上传图片并更新配置后，刷新浏览器页面 http://localhost:8000 查看效果。

## 💡 拍摄建议

### 全景图拍摄技巧
1. **保持相机水平**：使用三脚架保持相机稳定
2. **避免移动物体**：拍摄时室内最好无人
3. **充足光线**：打开所有灯，或在自然光充足时拍摄
4. **重叠拍摄**：如果手动拼接，相邻照片应重叠30%

### 普通图片拍摄技巧
1. **清理现场**：拍摄前收拾整齐
2. **多角度拍摄**：每个区域至少3个角度
3. **利用光线**：自然光最佳，避免逆光
4. **展示特色**：重点拍摄房屋亮点（落地窗、大空间等）

## ⚠️ 注意事项

1. **全景图是必须的**才能使用全景展示功能
2. 普通图片画廊功能，只要有图片就能正常显示
3. 图片命名建议使用**英文或拼音**，避免中文乱码
4. 确保图片**文件存在**后再更新配置，否则会显示加载失败
5. 图片文件大小建议控制在**2-5MB**以内，过大影响加载速度
