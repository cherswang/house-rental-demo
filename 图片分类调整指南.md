# 📷 图片分类调整指南

## 当前分类问题

您提到图片分类存在误差，将卫生间识别为了餐厅。由于我无法实际查看图片内容，需要您手动调整分类。

## 如何调整分类

### 方法1：修改脚本配置（推荐）

编辑 `script.js` 文件，找到 `galleryImages` 数组，修改对应图片的 `category` 值：

```javascript
{
    id: 3,
    src: 'assets/images/dining-room-1.jpg',
    thumb: 'assets/images/thumbs/dining-room-1.jpg',
    category: 'dining-room',  // 修改这里：'living-room', 'dining-room', 'bedroom', 'kitchen', 'bathroom'
    title: '餐厅全景',
    description: '现代简约餐厅'
}
```

### 方法2：重新命名图片文件

将图片文件重命名为正确的分类：

```bash
# 如果 dining-room-1.jpg 实际上是卫生间
mv dining-room-1.jpg bathroom-2.jpg
```

然后更新 `script.js` 中的配置。

## 分类列表

| 分类名称 | 对应房间 |
|---------|---------|
| `living-room` | 客厅 |
| `dining-room` | 餐厅 |
| `bedroom` | 卧室 |
| `kitchen` | 厨房 |
| `bathroom` | 卫生间 |
| `closet` | 衣帽间 |
| `balcony` | 阳台 |

## 手动调整步骤

1. 在浏览器中打开 http://localhost:8000
2. 查看每个分类下的图片
3. 如果发现分类错误，记录图片ID和正确的分类
4. 编辑 `script.js` 修改对应图片的 `category` 值
5. 刷新页面查看效果

## 示例

假设 `dining-room-1.jpg` 实际上是卫生间：

```javascript
// 修改前
{
    id: 3,
    src: 'assets/images/dining-room-1.jpg',
    category: 'dining-room',
    title: '餐厅全景'
}

// 修改后
{
    id: 3,
    src: 'assets/images/dining-room-1.jpg',
    category: 'bathroom',  // 改为卫生间
    title: '卫生间全景'
}
```

## 提示

- 您可以随时修改 `title` 和 `description` 来更好地描述图片内容
- 修改后刷新页面即可看到效果
- 如果需要添加新图片，只需在 `galleryImages` 数组中添加新对象即可
